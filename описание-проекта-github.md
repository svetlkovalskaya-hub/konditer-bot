# Описание проекта для GitHub

## Название

**Бот записи на торт для домашних кондитеров**

## Проблема

Домашние кондитеры часто ведут заказы в мессенджерах, блокнотах или голове. Из-за этого теряются детали, путаются даты выдачи, забываются телефоны и адреса доставки. Проект решает эту задачу — собирает заказы в одном месте, напоминает о ближайших выдачах и снимает рутину с кондитера.

## Пользователь

Проект создан для **домашних кондитеров и мастеров**, которые принимают заказы через Telegram и мессенджер MAX. Конечные пользователи — их клиенты, которые записываются на изделия через простой диалог с ботом.

## Основная функция

Сервис принимает заказы на кондитерские изделия через чат-бота и помогает кондитеру ими управлять.

**Что подаётся на вход:**
- Сообщение клиента в Telegram или MAX.
- Выбор изделия: торт, пряники, зефир, макарон, капкейки.
- Для тортов — выбор вида с фото и вкуса (например, Наполеон классический или шоколадный).
- Дата и время выдачи, способ получения, адрес или самовывоз, телефон, комментарий, фото-примеры.

**Что получается на выходе:**
- Сохранённый заказ в базе данных с номером и статусом.
- Уведомление кондитеру о новом заказе с пометкой источника: Telegram, MAX или ручная запись админа.
- Возможность переносить, искать и удалять заказы через админ-панель.
- Утреннее напоминание в 9:00 со списком заказов на сегодня и завтра.

**Задействованные инструменты:**
- Telegram-бот для клиентов.
- MAX-бот для интеграции с экосистемой MAX.
- Админ-панель для кондитера.
- SQLite-база для хранения заказов.

## Формат результата

- **Для клиента:** подтверждение заказа с номером и краткими деталями.
- **Для кондитера:** уведомления в Telegram, список заказов, поиск по имени и телефону, ручное создание заказа, перенос и удаление записей. Все данные хранятся локально в базе SQLite.

## Минимальный функционал (MVP)

Реализовано в первой версии:

- Запись клиента через Telegram-бота: выбор продукта, даты, времени, способа получения, телефона, комментария.
- Выбор вида торта с фото и вкуса для Наполеона и Медовика.
- Сохранение заказа в SQLite с пометкой источника.
- Уведомление админу о новом заказе.
- Админ-панель: просмотр списка, поиск по имени и телефону, удаление и перенос заказов.
- Ручное создание заказа от имени клиента.
- Утреннее приветствие с ближайшими заказами в 9:00.

**В планах:**
- Полный запуск MAX-бота.
- ИИ-помощник со списком продуктов для покупки.
- Напоминания о доставке и времени выдачи.
- Контент-агент для помощи в соцсетях и мессенджерах.
- Синхронизация с Google Calendar или Google Sheets.

## Технологии

- **Язык:** JavaScript (CommonJS).
- **Среда выполнения:** Node.js.
- **Telegram-бот:** `node-telegram-bot-api`.
- **MAX-бот:** `@maxhub/max-bot-api`.
- **База данных:** `better-sqlite3` (SQLite).
- **Веб-сервер:** `express` (healthcheck и сброс вебхука).
- **Планировщик:** `node-cron`.
- **Конфигурация:** `dotenv`.
- **Хостинг:** Beget VPS + PM2.
