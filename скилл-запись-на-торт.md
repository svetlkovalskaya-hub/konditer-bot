# Скилл: Telegram-бот для записи клиентов на торт

Этот скилл поможет развернуть такого же Telegram-бота для записи клиентов на кондитерские изделия. Подходит для домашних кондитеров и небольших мастерских.

## Что делает бот

- Принимает заказы через Telegram в виде пошагового диалога.
- Предлагает выбор изделий: торт, пряники, зефир, макарон, капкейки.
- Для тортов показывает список видов с фото и выбором вкуса (например, Наполеон классический или шоколадный).
- Сохраняет заказ в базе SQLite.
- Отправляет уведомление кондитеру.
- Даёт админ-панель для управления заказами: список, поиск, перенос, удаление, ручная запись.
- Утром в 9:00 присылает список заказов на сегодня и завтра.

## Стек

- JavaScript (CommonJS)
- Node.js
- `node-telegram-bot-api`
- `better-sqlite3`
- `express`
- `node-cron`
- `dotenv`

## Пошаговая установка

### 1. Создай бота в Telegram

1. Напиши [@BotFather](https://t.me/botfather).
2. Отправь `/newbot`.
3. Укажи имя и юзернейм бота.
4. Скопируй токен — он понадобится для `.env`.

### 2. Узнай свой Telegram ID

Напиши [@userinfobot](https://t.me/userinfobot) или [@getidsbot](https://t.me/getidsbot). Этот ID нужен для роли администратора.

### 3. Склонируй или создай проект

Если используешь этот репозиторий:

```bash
git clone https://github.com/svetlkovalskaya-hub/konditer-bot.git
cd konditer-bot
npm install
```

Если создаёшь с нуля — скопируй структуру из раздела выше и файлы из папки `src/bot/telegram`.

### 4. Настрой окружение

Создай файл `.env` рядом с `package.json`:

```env
TELEGRAM_BOT_TOKEN=токен_от_BotFather
ADMIN_TELEGRAM_ID=твой_telegram_id
PORT=3000
NODE_ENV=development
```

### 5. Добавь фото тортов

1. Создай папку `uploads/cake-photos/`.
2. Положи туда фото видов тортов.
3. Открой `src/config/cakeTypes.js` и укажи пути к файлам.

Пример:

```js
const CAKE_TYPES = [
  { name: 'Наполеон', photoFile: 'uploads/cake-photos/наполеон.jpg', flavors: ['Классический', 'Шоколадный'] },
  { name: 'Медовик', photoFile: 'uploads/cake-photos/медовик.jpg', flavors: ['Классический', 'Шоколадный'] },
  { name: 'Ваниль-клубника', photoFile: 'uploads/cake-photos/ваниль-клубника.jpg' },
];

module.exports = CAKE_TYPES;
```

Папка `uploads/` добавлена в `.gitignore`, поэтому на сервер фото нужно будет копировать отдельно.

### 6. Запусти бота локально

```bash
npm run dev
```

Если всё настроено правильно, в консоли появится:

```
Telegram-бот запущен
Сервер запущен на порту 3000
```

### 7. Проверь работу

1. Открой Telegram, найди своего бота.
2. Отправь `/start`.
3. Пройди весь сценарий: имя → изделие → дата → время → доставка/самовывоз → телефон → комментарий → фото.
4. Убедись, что админу пришло уведомление о заказе.
5. Проверь админ-панель через меню (доступно только для `ADMIN_TELEGRAM_ID`).

## Настройка лимитов и поведения

В `src/config/index.js` можно изменить:

- `maxPhotosPerOrder` — максимальное количество фото в заказе.
- `port` — порт сервера.

В `src/services/orderService.js` настрой:

- `MAX_ORDERS_PER_DATE` — максимальное число заказов на одну дату.

## Деплой на сервер

### Пример для Ubuntu / Debian

1. Установи Node.js 18+ или 22 LTS.
2. Склонируй проект на сервер.
3. Установи зависимости:

```bash
npm install
```

4. Пересобери нативные модули, если менялась версия Node.js:

```bash
npm rebuild better-sqlite3
```

5. Создай `.env` на сервере.
6. Скопируй фото тортов в `uploads/cake-photos/`.
7. Запусти через PM2:

```bash
npm install -g pm2
pm2 start src/index.js --name konditer-bot
pm2 save
pm2 startup
```

### Полезные команды

```bash
pm2 status                 # статус бота
pm2 logs konditer-bot      # логи
pm2 restart konditer-bot   # перезапуск
```

## Возможные проблемы

### Бот не отвечает

- Проверь токен в `.env`.
- Убедись, что у бота не активен webhook. Бот использует polling. Для сброса webhook можно открыть:

```
https://api.telegram.org/bot<TOKEN>/deleteWebhook
```

Или использовать эндпоинт `/reset-webhook`, если сервер запущен.

### Ошибки с базой данных

- Папка `data/` должна существовать и быть доступна для записи.
- Если менялась версия Node.js, пересобери `better-sqlite3` командой `npm rebuild better-sqlite3`.

### Фото тортов не показываются

- Проверь, что папка `uploads/cake-photos/` и файлы существуют на сервере.
- Проверь пути в `src/config/cakeTypes.js`.

### Утреннее приветствие приходит не в 9:00

- `node-cron` работает по системному времени сервера.
- Проверь часовой пояс сервера командой `timedatectl`.
