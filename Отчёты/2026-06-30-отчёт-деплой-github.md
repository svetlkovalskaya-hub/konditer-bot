## Следующий шаг

Деплой на Render. Нужно:
1. Зайти на https://render.com.
2. Создать New Web Service из репозитория `konditer-bot`.
3. Указать Build Command: `npm install`, Start Command: `npm start`.
4. Добавить переменные окружения:
   - `NODE_ENV=production`
   - `TELEGRAM_BOT_TOKEN=...`
   - `ADMIN_TELEGRAM_ID=...`
5. Добавить диск:
   - Name: `konditer-data`
   - Mount Path: `/data`
   - Size: 1 GB
6. Нажать Deploy.

После деплоя бот будет работать круглосуточно и не зависеть от локального VPN.
