# Отчёт: деплой бота на Beget VPS

**Дата:** 2026-07-02

## Что сделано

- Куплен VPS на Beget с Ubuntu 24.04.
- Установлены Node.js v18.19.1, npm, git.
- Склонирован репозиторий `svetlkovalskaya-hub/konditer-bot` в `/opt/konditer-bot`.
- Установлены зависимости через `npm install`.
- Создан `.env` с переменными окружения.
- Бот запущен через PM2 в фоновом режиме.
- Настроен автозапуск PM2 при перезагрузке сервера.

## Проверка

- Написать боту `/start` в Telegram.
- Пройти сценарий заказа.
- Проверить, создаётся ли заказ после нажатия «Да» на предпросмотре.

## Управление ботом на сервере

```bash
cd /opt/konditer-bot

pm2 status          # статус
pm2 logs            # логи
pm2 restart all     # перезапустить
pm2 stop all        # остановить
```

## Что нужно сделать дальше

1. Протестировать бота в Telegram.
2. Если есть баги — исправить и обновить код на GitHub, затем на сервере:
   ```bash
   cd /opt/konditer-bot
   git pull
   npm install
   pm2 restart all
   ```
3. Добавить автоматические напоминания и другие улучшения.
