# Отчёт: улучшения для админа и клиента

**Дата:** 2026-07-02

## Что сделано

### Для клиента

- Добавлен шаг ввода имени в начале сценария заказа.
- Добавлен шаг ввода контактного телефона.
- Имя и телефон отображаются в предпросмотре заказа.
- Телефон передаётся в уведомлении админу.

### Для кондитера

- Добавлена админская клавиатура с кнопками:
  - **📋 Список заказов**
  - **🔍 Поиск по имени**
- Список заказов сортируется по дате доставки/выдачи (от ближайшей к дальней).
- Добавлен поиск заказов по имени клиента.
- В деталях заказа добавлен телефон клиента.

### Технические изменения

- В таблицу `orders` добавлена колонка `phone`.
- Обновлён `orderService`: добавлен метод `searchOrdersByClientName`.
- Обновлены `newOrder.js`, `admin.js`, `index.js`, `keyboard.js`.

## Проверка

1. Создать новый заказ — убедиться, что спрашивает имя и телефон.
2. Зайти в бот как админ — убедиться, что есть кнопки «Список заказов» и «Поиск по имени».
3. Нажать «Список заказов» — заказы должны быть отсортированы по дате.
4. Нажать «Поиск по имени» — ввести имя и найти заказы.
