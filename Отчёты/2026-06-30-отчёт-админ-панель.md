# Отчёт: новая панель кондитера

## Что сделала

Переделала админскую часть бота с команд на кнопки. Теперь панель кондитера выглядит как простое меню, а не список `/orders`, `/order N` и прочего.

### Изменённые файлы

1. `src/utils/keyboard.js`
   - Обновила главную клавиатуру для админа.
   - Новые кнопки:
     - 📋 Список заказов
     - 🔍 Поиск по имени клиента
     - 📞 Поиск по номеру телефона
     - ❌ Удалить заказ
     - 🔄 Перенести запись на другое время
     - 🏠 Общее меню

2. `src/bot/telegram/admin.js`
   - Полностью переписала файл.
   - Убрала все slash-команды.
   - Добавила машину состояний для админа: поиск по имени, поиск по телефону, удаление заказа, перенос заказа.
   - Список заказов выводится по дате (ближайшие сверху), с разбивкой на порции по 5 штук.
   - Карточка заказа показывает: номер, статус, имя, телефон, Telegram, изделие, дату, время, адрес/самовывоз, комментарий.
   - Удаление — мягкое, заказ переводится в статус `cancelled` и пропадает из активных списков.
   - Перенос просит номер заказа, затем новую дату в формате `ГГГГ-ММ-ДД`.

3. `src/services/orderService.js`
   - Добавила `searchOrdersByPhone(phone)`.
   - Добавила `deleteOrder(orderId)` для мягкого удаления.

4. `src/bot/telegram/index.js`
   - Приветствие админа теперь чистое: «Привет! Это панель кондитера. Выберите действие».
   - Убрала старый список команд из приветствия.
   - Обновила меню команд бота: `/start` → «Открыть меню».
   - Убрала регистрацию старых slash-команд (`admin.register`).
   - Добавила обработку новых кнопок и админских состояний.
