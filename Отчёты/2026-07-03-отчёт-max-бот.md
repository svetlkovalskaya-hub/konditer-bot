# Отчёт: добавлен MAX-бот записи на торт

**Дата:** 2026-07-03

## Что сделано

Создан MAX-бот, полностью повторяющий сценарий Telegram-бота для записи клиентов на торты и другие изделия.

## Изменённые файлы

1. `package.json` / `package-lock.json` — добавлена библиотека `@maxhub/max-bot-api`.
2. `.env.example` — добавлена переменная `MAX_BOT_TOKEN`.
3. `src/config/index.js` — добавлен `maxBotToken`.
4. `src/utils/maxKeyboard.js` — новый модуль с клавиатурами в формате MAX API.
5. `src/bot/max/index.js` — полноценный MAX-бот с сценарием заказа.
6. `src/index.js` — Telegram-бот передаётся в MAX-бот для уведомлений администратора.

## Сценарий MAX-бота

1. Приветствие с кнопками «📝 Записаться» и «📋 Мои заказы».
2. Ввод имени.
3. Выбор продукта (Торт, Пряники, Зефир и т.д.).
4. При выборе «Торт» — список видов тортов в алфавитном порядке.
5. Просмотр фото выбранного торта.
6. Для Наполеона и Медовика — выбор вкуса «Классический / Шоколадный».
7. Подтверждение выбора.
8. Выбор даты через inline-календарь.
9. Выбор времени.
10. Доставка или самовывоз.
11. Ввод телефона и комментария.
12. Прикрепление фото-примеров (до 5 штук).
13. Предпросмотр заказа и подтверждение.
14. Сохранение заказа в общую базу SQLite.
15. Уведомление администратора в Telegram с пометкой `(MAX)`.

## Что нужно для запуска MAX-бота

1. Получить токен бота в личном кабинете MAX для разработчиков: https://dev.max.ru/docs/chatbots/bots-coding/prepare
2. Добавить в файл `.env` на сервере:
   ```
   MAX_BOT_TOKEN=your_max_bot_token_here
   ```
3. Перезапустить бота:
   ```bash
   cd /opt/konditer-bot
   pm2 restart all
   ```
